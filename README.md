# NetWave Digital

A professional software development company website built with modern web technologies.

## About

NetWave Digital is a leading software development, web design, mobile app development, and digital marketing company based in Bokaro Steel City, Jharkhand, India.

## Tech Stack

- **Frontend**: React with TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Build Tool**: Vite
- **Routing**: React Router

## Getting Started

1. Install dependencies:
```sh
npm install
```

2. Start the development server:
```sh
npm run dev
```

3. Build for production:
```sh
npm run build
```

## Project Structure

- `src/components/` - React components
- `src/pages/` - Page components
- `src/assets/` - Images and static assets
- `src/hooks/` - Custom React hooks
- `src/lib/` - Utility functions
